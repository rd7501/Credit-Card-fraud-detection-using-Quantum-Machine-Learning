# Credit Card Fraud detection Using QML

This repository contains a Quantum Machine Learning (QML) approach for credit card fraud detection. It includes notebooks, model artifacts, and related files.

Quick start
- Initialize git (already done locally by the helper script).
- Create a GitHub repository and add the remote (see below).

To create a remote and push (example):

```powershell
Set-Location -LiteralPath 'c:\Users\ramdw\Desktop\Credit Card Fraud detection Using QML'
# If using GitHub CLI (authenticated):
# gh repo create <owner>/<repo> --public --source . --push

# Or add remote manually and push:
git branch -M main
git remote add origin https://github.com/<your-username>/<repo>.git
git push -u origin main
```
